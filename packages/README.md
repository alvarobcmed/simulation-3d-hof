# Shared Packages

This directory contains shared libraries and packages used across the platform.

## Packages

### Shared Types (`shared-types/`)
- Common data types and interfaces
- API contract definitions
- Cross-platform type safety
- Protocol buffer definitions

### Common Utils (`common-utils/`)
- Utility functions and helpers
- Logging and error handling
- Configuration management
- Validation schemas

### SDK (`sdk/`)
- Client SDKs for platform integration
- API client libraries
- Authentication helpers
- Platform-specific implementations

## Usage

Packages are designed to be consumed by:
- iOS application
- Backend services
- External integrations
- Third-party developers

## Development

Each package follows semantic versioning and has its own:
- Build process
- Test suite
- Documentation
- Release cycle

## Publishing

Packages are published to:
- Internal package registries
- Public registries (where appropriate)
- Platform-specific package managers
