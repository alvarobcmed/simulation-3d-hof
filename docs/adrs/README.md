# Architecture Decision Records (ADRs)

Este diretório contém os Architecture Decision Records (ADRs) do projeto Simulation 3D HOF.

## Formato

Cada ADR segue o template:
- **Status**: Proposto | Aceito | Rejeitado | Superseded
- **Context**: Contexto e motivação para a decisão
- **Decision**: A decisão tomada
- **Consequences**: Consequências positivas e negativas

## Processo de Criação

1. Copie o template `ADR-template.md`
2. Renomeie para `ADR-XXX-descriptive-name.md`
3. Preencha todas as seções
4. Crie um PR para discussão
5. Após aprovação, atualize o status para "Aceito"
