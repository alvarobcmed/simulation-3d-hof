# Applications

This directory contains the client applications for the 3D Orofacial Harmonization Simulation platform.

## Applications

### iOS App (`ios-app/`)
- Swift-based iOS application
- ARKit integration for 3D capture and AR mirror
- Core user interface and simulation features
- Offline capability support

### Admin Dashboard (`admin-dashboard/`)
- Web-based administration interface
- User management and analytics
- System monitoring and configuration
- CRM/EMR integration management

## Development

Each application has its own development setup and build process. Refer to the individual README files in each application directory for specific instructions.
